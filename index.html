What We Do

In order to prepare young learners for their future mathematical and computer science courses, we publish articles & videos on the topics featured below:

Mathematics

Math is commonly used in the world of computer science. The most common types of math used in this field are advanced algebra and precalculus.

Programming

Programming lies at the heart of computer science. Programming languages featured in this project are HTML/CSS/JS, PHP, MySQL, and Python/Micropython.

Hardware

Software cannot exist without proper hardware. This project will teach you about various types of computer hardware from microcontrollers to single board computers (SBC's).

Raspberry Pi Pico Webserver

Graphical Applications in Python

Learn the basics of Tkinter, a python module that is commonly used to create GUI applications.

Prerequisite: Geometry, Algebra II
